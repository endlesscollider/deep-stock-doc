# 策略说明

## 可用策略

| 策略 | 说明 |
|------|------|
| `top5` | 买入预测收益前5的股票 |
| `top10` | 买入预测收益前10的股票 |
| `top20` | 买入预测收益前20的股票 |
| `long` | 只做多策略 |
| `momentum` | 动量策略 |
| `reversion` | 反转策略 |

## 策略详解

### Top-K 策略

选择预测收益最高的前 K 只股票，等权买入。

### 动量策略

买入近期涨幅较大的股票。

### 反转策略

买入近期跌幅较大的股票，赌反转。

## 使用示例

```bash
python examples/04_backtest.py --strategy top10 --start-date 20250101 --end-date 20250331
```
